# RPG Batalha por Turnos

## Overview
A classic turn-based RPG battle game built with pure HTML, CSS, and JavaScript. Fully in Portuguese (Brazilian), playable directly in any modern web browser with no installation required.

## Project Structure
- `index.html` - The complete game (HTML + CSS + JavaScript all in one file)
- `teste-balanceamento.js` - Testing/simulation script for game balance analysis
- `README.md` - Project documentation

## How to Run
The project is served as a static website using Python's built-in HTTP server on port 5000.

## Game Features
- Turn-based battle system with hit chance based on attributes
- 50+ types of monsters with unique stats and special effects
- Status/debuff system: poison, paralysis, attack/defense buffs
- Consumable inventory and item shop in the city
- Gold economy from battles for healing and item purchases
- Hero customization with attribute evolution per level
- Game over screen with final statistics and new game option
- Responsive layout for desktop and mobile

## Technical Details
- Pure HTML5/CSS3/JavaScript (no frameworks or build tools)
- All game logic contained in the single index.html file
- No external dependencies
